Each type creates an output file w columns:

* `platform`
* `platform_id`
* `name`
* `website`
* `additional` (may contain website, social, etc. info)

Types include:

* [Tally](https://www.tally.xyz/)
* [Realms](https://realms.today/)
* Snapshot
